# Contributing to KittenSushiGirl

🎉 Thanks for wanting to contribute! (Or at least, thanks for trying.)

## How It Works

This project uses an **AI-first workflow**:

1. Pick an issue from the board
2. Create a branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Open a PR with `Closes #123`
5. Auto-merge when tests pass

## Labels We Use

- `bug` — Something's broken
- `enhancement` — New feature or improvement
- `priority:high` — Do this first
- `priority:medium` — Do this second  
- `priority:low` — Nice to have

## Getting Started

```bash
git clone https://github.com/PatrickBamber/kittensushigirl-poc
cd kittensushigirl-poc
npm install
npm run dev
```

## Running Tests

```bash
npm run test
```

## Code Style

- Keep it fun but readable
- Use emoji in commits 🎉
- Don't take yourself too seriously
